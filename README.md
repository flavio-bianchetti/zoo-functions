# Zoo Functions

Este repositório contém o projeto "_Zoo Functions_", requisito em atendimento ao módulo de Fundamentos - Higher Order Functions do JavaScript ES6, da Trybe. O projeto consiste em implementar várias funções para atender aos requisitos propostos e garantir que todas as funções passem nos testes unitários.

### Habilidades:
 - Produzir código legível, conciso e expressivo utilizando as novas funcionalidades do ES6;
 - Utilizar as Higher Order Functions para manipular e criar arrays;
 - Escolher a Higher Order Function mais adequada para a obtenção de um resultado esperado;
 - Aprender a usar de forma conjunta as Higher Order Functions;
 - Interpretar testes unitários e produzir soluções que atendam a eles.
